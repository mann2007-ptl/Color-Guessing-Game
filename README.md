
# 🎨 Color Guessing Game

A simple and interactive **Color Guessing Game** built using **HTML, CSS, and JavaScript**.  
The goal of the game is to correctly identify a color based on its **RGB value**.

---

## 🚀 Features

- 🎯 Guess the correct color using RGB values  
- 🔥 Current streak tracking  
- 🏆 Best streak tracking  
- ❤️ Lives system for limited attempts  
- 🎮 Difficulty modes:
  - Easy (fewer colors)
  - Hard (more colors)  
- 🔄 New Round functionality  
- ♻️ Reset streak option  
- ✨ Clean and modern UI  

---

## 🛠️ Tech Stack

- HTML  
- CSS  
- JavaScript (Vanilla JS)

## 🎯 How to Play

1. A random **RGB color value** is displayed at the top  
2. Multiple colored boxes are shown  
3. Click on the box that matches the RGB value  

### ✔️ If Correct:
- Streak increases  

### ❌ If Wrong:
- You lose one life  

👉 Try to achieve the **highest streak possible**
